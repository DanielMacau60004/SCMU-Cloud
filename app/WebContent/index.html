Root for SCC backend

Media:

  • [POST] /rest/media/ Upload a file
  • [POST] /rest/media/user/{id} Change the user photo
  • [POST] /rest/media/house/{id} Add a photo to a house
  • [GET] /rest/media/{id} Download a file
  • [LIST] /rest/media/ List all files

Users:

  • [POST] /rest/user/auth User auth
  • [POST] /rest/users/ Create a new user
  • [PUT] /rest/users/{id} Update a user
  • [GET] /rest/users/{id} Get a specific user
  • [GET] /rest/users/ List all users

Houses:

  • [POST] /rest/houses/ Create a new house
  • [PUT] /rest/houses/{id} Update a house
  • [GET] /rest/houses/{id} Get a specific house
  • [GET] /rest/houses/ List all houses
  • [GET] /rest/houses/user/{id} List houses by user
  • [GET] /rest/houses/location List houses by location
  • [GET] /rest/houses/interval List houses by location and period
  • [GET] /rest/houses/filter Filter all houses by the name and description
  • [GET] /rest/houses/popular List all houses with more rentals
  • [GET] /rest/houses/recent List all houses that were recently added

Promotions:

  • [POST] /rest/houses/promotion/{houseId} Create a new promotion

Questions:

  • [POST] /rest/houses/{houseId}/question Create a new question
  • [PUT] /rest/houses/{houseId}/question/{id} Answer a question
  • [GET] /rest/houses/{houseId}/question List all questions

Rentals:

  • [POST] /rest/houses/{houseId}/rental Create a new rental
  • [PUT] /rest/houses/{houseId}/rental/{id} Update a rental
  • [GET] /rest/houses/{houseId}/rental Get a specific rental
  • [GET] /rest/houses/{houseId}/rental List all rentals
  • [GET] /rest/houses/promotions List discounted rentals in the near future
